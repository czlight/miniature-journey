#Document what I remember about vim

:q - quit
:w - save
i - enter insert mode
<Esc> - exit insert mode (enter command mode)
k - move up
j - move down
l - move right
h - move left

represented as:
	k
    h	   l
	j
set list
set listchars
:x - quit without saving

:help help_file.txt - replace help_file.txt with actual help fil
set listchars=eol:!
